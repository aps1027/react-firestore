# Sample React App with Firestore

## Technologies
1. React.js
1. Firebase (firestore)

## Features
1. Process Registration
1. Product Edition
1. Product Deletion
1. Product list

## Routes
1. Register (`<http://localhost:3000>/register`)
1. Edit (`<http://localhost:3000>/register`)
1. List (`<http://localhost:3000>`)

## Development
1. Run `npm install`
1. Run `npm start`
